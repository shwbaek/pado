## âš™ï¸ Installation

You can install PADO via pip:

```bash
pip install pado
```

Or install directly from GitHub:

```bash
pip install git+https://github.com/shwbaek/pado.git
```

For development installation:

```bash
git clone https://github.com/shwbaek/pado.git
cd pado
pip install -e .
```
